# PPO6
```
Программа принимает выражение в виде строки, делит ее на токены, переводит в обратную польскую запись и вычисляет выражение.
Когда она принимает пустую строку, то прием строк останавливается.
В папке test 2 теста 
1ый принимает выражение в виде строки производит все действия и сравнивает результат с ожидаемым ответом
2ой проверяет обработку ошибок
```
